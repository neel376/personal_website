Overview

Methodology

Sonic Remake was made using the Unity 3D engine. This game's goal just like all Sonic games is to reach the end of the stage as fast as possible.

This game currently features 3 playable character: Sonic, Shadow, and tails, each having different stats and wielding their unique special move.

I started this game near the end of 2013 during my junior year of high school and plan on working on it again in the future.

Setting up the Sprites

I decided to use sprites to capture the original sonic. With photoshop, I was able to extract individual sprites from a sprite sheet. Then using Unity's built in Sprite editor, I was able to take the sprites and convert them into usable game objects.
